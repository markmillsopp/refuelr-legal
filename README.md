This repository holds a minimal GitHub Pages site for Refuelr: Home, Privacy Policy and Terms of Service.

How to publish:

1. Create a new public GitHub repository named `refuelr-legal` under the `markmillsopp` account (or your preferred account).
2. Clone the repo locally and copy the files from the workspace `refuelr-legal` folder into it.
3. Commit and push to `main`.
4. In the repository Settings → Pages select source: `main` branch → `/ (root)` or `/docs` if you prefer.
5. Add the custom domain `refuelr.co.uk` in Pages settings and update DNS as described in the `docs/README.md` in the main repo.

Notes:
- Ensure the repo is public for free GitHub Pages HTTPS.
- If you want me to push this repo for you, provide a remote URL and confirm you want commits authored as `mark.millsopp <mark.millsopp@gmail.com>`.
